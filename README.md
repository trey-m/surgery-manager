# Surgery Manager

- View a list of surgeries
- Schedule a new surgery
- Cancel an upcoming surgery

# Architecture

- [Fastify/Vite SSR](https://fastify-vite.dev/)
- [React](https://react.dev/)
- [SQLite](https://www.sqlite.org/)
- [TailwindCSS](https://tailwindcss.com/)
